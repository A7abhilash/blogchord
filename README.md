# Blogchord

> A blogging platform designed for little stories that make your day better.

### Features

1. Post/Edit your blogs
2. Set private or public status for your blogs
3. User Profile Visit
4. Save posts(Bookmark)
5. Likes
6. Alerts(in site)

### Technologies

1.Stack: MERN
